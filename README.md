# Sentinel Agents

Constraint-first governed runtime for autonomous AI agents.

## What it is

Sentinel Agents is a prototype runtime for agent execution under policy, risk, memory, evaluation, governance, deployment, and self-governance constraints.

It is designed to explore how autonomous systems can be made more controlled, traceable, and adaptable under uncertainty.

## Current status

Implemented at prototype level:

- Phase 1: Core runtime
- Phase 2: Secure execution
- Phase 3: Risk intelligence
- Phase 4: Swarm coordination
- Phase 5: Memory layer
- Phase 6: Adaptive learning
- Phase 7: Evaluation layer
- Phase 8: Governance and audit
- Phase 9: Deployment layer
- Phase 10: Self-governance layer
